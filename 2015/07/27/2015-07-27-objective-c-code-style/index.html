谈谈Objective-C编码规范

发表于 2015-07-27   |  

说到代码规范，网上有一些资料，但是很少会遵循代码规范的。当别人看我们完成某些功能时，如果自己感觉代码，不忍直视的时候，想想别人看到时候的感受吧。遵守好的代码规范会带来很多好处：

  • 促进团队合作
  • 减少bug处理
  • 降低维护成本
  • 方便代码审查
  • 便于开发人员自身的成长
